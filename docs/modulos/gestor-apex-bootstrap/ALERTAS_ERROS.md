📖 Visão Geral do Projeto
O Apex Gestor é uma solução de software robusta, pensada para atender às necessidades específicas de uma loja de material de construção. O sistema centraliza informações e automatiza tarefas, desde o controle de entrada e saída de produtos até a geração de relatórios gerenciais, proporcionando uma visão 360° da empresa.

✨ Funcionalidades Principais
- Central de pendências com cards de falha de conexão, estoque baixo e permissão negada.
- Botão "Tentar novamente" entra em estado de carregamento, fica desabilitado e depois informa se a conexão voltou.
- Botão "Repetir sincronização" simula nova tentativa segura sem travar a operação do caixa.
- Botão "Gerar pedido de compra" cria rascunho de reposição para revisão do gerente.
- Botão "Solicitar liberação" orienta o operador e envia solicitação simulada para perfil autorizado.
- Mensagens de erro explicam o problema em linguagem clara e indicam quando o usuário pode resolver sozinho.

🚀 Tecnologias Utilizadas

Front-end: HTML, CSS, Bootstrap

Backend: Java (Servlets & JSP)

Banco de Dados: PostgreSQL

Servidor de Aplicação: Apache Tomcat

Controle de Versão: Git & GitHub

Gerente de Dependências: Maven
