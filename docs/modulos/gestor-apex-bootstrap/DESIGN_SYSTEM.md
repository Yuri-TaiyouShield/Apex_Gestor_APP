📖 Visão Geral do Projeto
O Apex Gestor é uma solução de software robusta, pensada para atender às necessidades específicas de uma loja de material de construção. O sistema centraliza informações e automatiza tarefas, desde o controle de entrada e saída de produtos até a geração de relatórios gerenciais, proporcionando uma visão 360° da empresa.

✨ Funcionalidades Principais
- Design System Bootstrap com botões primários, secundários, ghost, perigo suave e botões grandes de produto para PDV.
- Suporte nativo a Light Mode e Dark Mode por `data-bs-theme`, com persistência em `localStorage`.
- Estados de botão definidos para normal, hover, active, focus-visible e disabled, garantindo que toda ação tenha feedback claro.
- Toasts e modais padronizados para confirmações, erros recuperáveis e mensagens operacionais.
- Paleta profissional com Azul Petróleo Profundo, cinzas suaves, Branco Neve e cores de status para sucesso, atenção e erro.
- Componentes pensados para futura conversão em JSP, mantendo IDs semânticos e estilos reutilizáveis.

🚀 Tecnologias Utilizadas

Front-end: HTML, CSS, Bootstrap

Backend: Java (Servlets & JSP)

Banco de Dados: PostgreSQL

Servidor de Aplicação: Apache Tomcat

Controle de Versão: Git & GitHub

Gerente de Dependências: Maven
